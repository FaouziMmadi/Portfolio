Site du Centre social de Malpasse Creation un site statique du centre social

Description du projet

Travaille d'équipe effectuer en stage pendant la formation de Passerelle Numérique

Le but était de faire un site statique pour le centre social de Malpasse. Elle n'avait pas de site au départ donc nous avons commencé par lui presenter des maquettes pour lui donner une idée de site internet.

Une fois les maquettes finies nous sommes allé voir la directrice pour quel nous donnes son avis. Elle a décidé de mélanger les 2 maquettes que vous voyez sur la page.

Nous avons hébergé le site web sur OVH puis nous avons mis le site directement en ligne. Sur la dernière semaine nous avons mis le texte que la directrice nous a fourni et nous avons corrigé tous les derniers petits détails.
